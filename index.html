Example 4 - Wheelzoom & Grid

You can zoom with your mouse wheel. Since gridShow is set and either gridX or gridY is set to a value greather than 1, swoosh will draw a grid useing a ::before pseudo element. Drag the picture around, and you will see that it will always fit to the grid or to the edges.

Wheelzoom & Grid

swoosh(document.getElementById("wheelzoom"), {
  gridX: 100,
  gridY: 50,
  gridShow: true,
  wheelScroll: false,
  wheelZoom: true,
});

Example 6 - Events & Callbacks

Events & Callbacks

There are 4 events emitted by swoosh. You can attach and detach custom callbacks to those events. To realise an infinity scroll element, you can load and append additional content when collide.bottom triggers.

var log = document.getElementById("log");
swoosh(document.getElementById("events"))
  .on('collide.left', function(e){
    log.innerHTML += "collide.left Event triggered<br>"
  })
  .on('collide.top', function(e){
    log.innerHTML += "collide.top Event triggered<br>"
  })
  .on('collide.right', function(e){
    log.innerHTML += "collide.right Event triggered<br>"
  })
  .on('collide.bottom', function(e){
    log.innerHTML += "collide.bottom Event triggered<br>"
  });

Example 7 - Anchor Links

Anchor Links

This is the anchor with id="anchors_top".
Anchor links inside a swoosh element are handled by the swoosh element itself. This behavior can be disabled with handleAnchors=false.
However, it will not trigger a hashchange event, even when the hash is changed in the address bar of the browser. It is done using History.pushState(). Swoosh makes sure, that every element which is involved (which is an ancestor to the anchor element), will scroll to the place of the anchor or to the inner swoosh element which contains the anchor.

Public Accessible Methods

Some methods are public accessible and can be invoked outside the object context. Click on the links to see how to use them.
  • scrollTo(): swoosh.scrollTo(100, 100)
  • scrollBy(): swoosh.scrollBy(50, 50)
  • scaleTo(): swoosh.scaleTo(2)
  • scaleBy(): swoosh.scaleBy(90)
  • reinit(): Reinitialize with current config. swoosh.reinit()
  • on(): See Events & Callbacks
  • off(): See Events & Callbacks
  • destroy(): Revert all DOM manipulations and deregister all event handlers. swoosh.destroy()
  • Anchor Links: top anchor, right anchor, bottom anchor
